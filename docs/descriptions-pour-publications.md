### 4) Long (2 paragraphes) — article, thread forum, présentation détaillée

Le projet **FiveM RP Community Hub** (repo *harrata-city* sur GitHub) fournit un site vitrine complet pour une communauté FiveM : navigation structurée, pages éditables, forum intégré, formulaires de candidature, grille boutique compatible Tebex, galerie et actualités avec pages détail. L’accent est mis sur la **maintenabilité** : une grande partie du contenu se pilote depuis l’admin (stockage local / export JSON selon ta configuration), avec possibilité d’**API** pour le statut live du serveur (Cfx.re ou endpoint `players.json`).

L’authentification joueurs/staff repose sur **OAuth Discord** et **Steam**. La démo publique illustre le rendu ; ce n’est pas une annonce de serveur RP à rejoindre, mais un **template** à forker. Documentation dans le README du dépôt ; support et échanges pendant la bêta sur le **Discord du projet** : https://discord.gg/7XVtaRnpQe

### 4) Long (two paragraphs) — blog post, detailed forum thread

The **FiveM RP Community Hub** project (*harrata-city* on GitHub) ships a full marketing site for a FiveM community: structured navigation, editable pages, built-in forum, application forms, Tebex-friendly shop grid, gallery and news with detail routes. The focus is **maintainability**: much of the content is driven from the admin (local storage / JSON export depending on setup), with optional **API** integration for live server status (Cfx.re or `players.json`).

Technically it is a **Next.js 15** (App Router), **React 19**, **TypeScript** and **Tailwind CSS 4** app, deployable on **Vercel** or any Node 20+ host. Player/staff authentication uses **Discord OAuth** and **Steam**. The public demo showcases the UI; it is **not** a “join my server” ad but a **template** to fork. See the repository README; beta support and discussion on the project **Discord**: https://discord.gg/7XVtaRnpQe
