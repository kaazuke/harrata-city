# Bannière d'accueil

Petite extension de démo qui ajoute une bannière personnalisable en haut du site.

## Configuration

Modifiez les `settings` depuis l'admin :

| Clé           | Type      | Description                                            |
| ------------- | --------- | ------------------------------------------------------ |
| `message`     | `string`  | Texte affiché dans la bannière                         |
| `color`       | `string`  | Couleur d'accent (hex)                                 |
| `ctaLabel`    | `string`  | Libellé du bouton (vide = pas de bouton)               |
| `ctaUrl`      | `string`  | URL du bouton                                          |
| `dismissible` | `boolean` | Si vrai, l'utilisateur peut fermer la bannière         |

## Installation

1. Téléchargez `welcome-banner.zip`
2. Allez dans Admin → Extensions
3. Glissez-déposez le zip dans la zone d'import
4. L'extension est activée immédiatement
